\section*{2. Стационарное решение при отсутствии поглощения}

\hrule\vspace{5mm}

Исключим затухание и положим все производные равными нулю:
\begin{equation}
\left\{
\begin{array}{ccl}
0 &=& - d m_y + bF n_x\\
0 &=& h m_z + d m_x + bF n_y\\
0 &=& - m_y h + bFn_z\\
0 &=& - 2n_y(1+d) - n_y(m_z-1) + n_zm_y + C(m_x-h)\\
0 &=&   2n_x(1+d) - n_z(m_x-h) + n_x(m_z-1) + Cm_y\\
0 &=& - n_xm_y + n_y(m_x-h) + C(m_z-1)\\
0 &=& n_x(m_x-h) + n_ym_y + n_z(m_z-1)
\end{array}\right.
\end{equation}

Из первого и третьего уравнения $m_y = n_x bF/d, \quad n_z = n_x h/d$
Исключим $n_z$ и $m_y$ из остальных уравнений:
\begin{equation}
\left\{
\begin{array}{ccl}
0 &=&   h m_z  +  d m_x + bFn_y\\
0 &=&   n_x(m_x-h) + n_yn_x bF/d + n_x h/d (m_z-1)\\
0 &=& - 2n_y(1+d) - n_y(m_z-1) + n_x^2 bFh/d^2 + C(m_x-h)\\
0 &=&   2n_x(1+d) - n_x (m_x-h)h/d + n_x(m_z-1) + C n_x bF/d\\
0 &=& - n_x^2 bF/d + n_y(m_x-h) + C(m_z-1)\\
\end{array}\right.
\end{equation}

Из второго уравнения
$$
0 = \frac{n_x}{d} (hm_z + dm_x + bFn_y - dh  - h).
$$

Подставив сюда первое уравнение, получим, что $n_x=0$.
Отсюда

\begin{equation}
m_y = n_z = n_x=0,\quad n_y=\pm1
\end{equation}

и уравнения принимают вид:

\begin{equation}
\left\{
\begin{array}{ccl}
0 &=&   h m_z  +  d m_x + bn_yF\\
0 &=&   2(1+d) + (m_z-1) - n_yC(m_x-h)\\
0 &=&   m_x - h + n_yC(m_z-1)\\
\end{array}\right.
\end{equation}

(здесь я воспользовался тем, что $n_y n_y = 1$).

\vspace{5mm}
\hrule\vspace{5mm}
Видно, что при $h,d\to0$ решение первого уравнения~--- $F=0$, то есть
$\theta = \pm\arccos(-1/4)$. Введем числа $s_1 = \pm1$~--- знак $n_y$,
$s_2 = \pm1$~--- знак $\theta$ и $s=s_1 s_2$.

Будем искать решение в виде  $\theta = s_2\left[\arccos(-1/4) +
\Theta\right]$, где $\Theta \sim h,d\ll 1$. Пренебрегая членами,
квадратичными по $\Theta$, получим:

\begin{equation}
  \sin(\theta) = s_2 \frac{\sqrt{15}-\Theta}{4}, \qquad
  \cos(\theta) = - \frac{1 + \sqrt{15}\Theta}{4}
\end{equation}

\begin{equation}
  C = \ctg(\theta/2) = s_2\sqrt{\frac{1+\cos\Theta}{1-\cos\Theta}} =
s_2\left(\sqrt{\frac35} - \frac45\Theta\right), \quad
  F = \frac{4}{15}\sin{\theta}(1+4\cos{\theta}) = -s_2\Theta
\end{equation}
\hrule\vspace{5mm}

Подставляем $C$ и $F$ в уравнения:

\begin{equation}
\left\{
\begin{array}{ccl}
0 &=&   h m_z  +  d m_x - s b\Theta\\
0 &=&   2(1+d) + (m_z-1) - s \left(\sqrt{\frac35} - \frac45\Theta\right)(m_x-h)\\
0 &=&   m_x - h + s\left(\sqrt{\frac35} - \frac45\Theta\right)(m_z-1)\\
\end{array}\right.
\end{equation}

Из второго уравнения
\begin{equation}
m_z =  - 1 - 2d + s\sqrt{\frac35}(m_x-h) - s \frac45\Theta m_x
\end{equation}

\begin{equation}
\left\{
\begin{array}{ccl}
m_x (d + s \sqrt{\frac35}h) &=& h + s b\Theta\\
m_x (1-\sqrt{\frac35} \Theta) &=& h
+ s \frac{\sqrt{15}}4 (1+d)
- s \Theta
\end{array}\right.
\end{equation}

\begin{equation}
\frac{h + s b\Theta}{d + s \sqrt{\frac35}h}(1-\sqrt{\frac35} \Theta) =
 h + s \frac{\sqrt{15}}4 (1+d) - s \Theta
\end{equation}

\begin{equation}
h + s b\Theta =  s \frac{\sqrt{15}}4\left(d + s \sqrt{\frac35}h\right)
\end{equation}

\begin{equation}
\Theta = \frac1{4b}\left(\sqrt{15}d - s h\right), \qquad
m_x  = h + s \frac{\sqrt{15}}4 (1+d) - s \Theta, \qquad
m_z  = - \frac14 - \frac54 d - 2\sqrt{\frac35} \Theta
\end{equation}

\noindent\fbox{\parbox{\textwidth}{
Стационарное решение уравнения Леггетта во вращающейся системе
координат для малых $h$ и $d$:
$$
n_x=0, \qquad
n_y=s_1, \qquad
n_z=0,
$$

$$
\Theta = \frac1{4b}\left(\sqrt{15}d - s h\right),\qquad
\theta=s_2 [\arccos(-1/4) + \Theta],
$$

$$
m_x  = h + s\frac{\sqrt{15}}4 (1+d) - s\Theta, \qquad
m_y = 0, \qquad
m_z  = - \frac14 - \frac54 d - 2\sqrt{\frac35} \Theta
$$
