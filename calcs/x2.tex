\section*{2. Стационарное решение при отсутствии поглощения}

\hrule\vspace{5mm}

Исключим затухание и положим все производные равными нулю:
\begin{equation}
\left\{
\begin{array}{ccl}
0 &=& - d m_y + bF n_x\\
0 &=& h m_z + d m_x + bF n_y\\
0 &=& - m_y h + bFn_z\\
0 &=& - 2n_y(1+d) - n_y(m_z-1) + n_zm_y + C(m_x-h)\\
0 &=&   2n_x(1+d) - n_z(m_x-h) + n_x(m_z-1) + Cm_y\\
0 &=& - n_xm_y + n_y(m_x-h) + C(m_z-1)\\
0 &=& n_x(m_x-h) + n_ym_y + n_z(m_z-1)
\end{array}\right.
\end{equation}

Из первого и третьего уравнения $m_y = n_x bF/d, \quad n_z = n_x h/d$
Исключим $n_z$ и $m_y$ из остальных уравнений:
\begin{equation}
\left\{
\begin{array}{ccl}
0 &=&   h m_z  +  d m_x + bFn_y\\
0 &=&   n_x(m_x-h) + n_yn_x bF/d + n_x h/d (m_z-1)\\
0 &=& - 2n_y(1+d) - n_y(m_z-1) + n_x^2 bFh/d^2 + C(m_x-h)\\
0 &=&   2n_x(1+d) - n_x (m_x-h)h/d + n_x(m_z-1) + C n_x bF/d\\
0 &=& - n_x^2 bF/d + n_y(m_x-h) + C(m_z-1)\\
\end{array}\right.
\end{equation}

Из второго уравнения
$$
0 = \frac{n_x}{d} (hm_z + dm_x + bFn_y - dh  - h).
$$

Подставив сюда первое уравнение, получим, что $n_x=0$.
Отсюда

\begin{equation}
m_y = n_z = n_x=0,\quad n_y=\pm1
\end{equation}

и уравнения принимают вид:

\begin{equation}
\left\{
\begin{array}{ccl}
0 &=&   h m_z  +  d m_x \pm bF\\
0 &=&  2(1+d) + (m_z-1) \mp C(m_x-h)\\
0 &=&  (m_x-h) \pm C(m_z-1)\\
\end{array}\right.
\end{equation}

\hrule\vspace{5mm}
Видно, что при $h,d\to0$ решение первого уравнения~--- $F=0$,
то есть $\theta = \pm\arccos(-1/4)$.

Будем искать решение в виде
$\theta = \pm\left[\arccos(-1/4) + \Theta\right]$ при $\Theta \sim h,d\ll 1$.

Пренебрегая членами, квадратичными по $\Theta$, получим:

\begin{equation}
  \sin(\theta) = \pm \frac{\sqrt{15}-\Theta}{4}, \qquad
  \cos(\theta) = - \frac{1 + \sqrt{15}\Theta}{4}
\end{equation}

\begin{equation}
  C = \ctg(\theta/2) = \pm\sqrt{\frac{1+\cos\Theta}{1-\cos\Theta}} =
\pm\left(\sqrt{\frac35} - \frac45\Theta\right), \quad
  F = \frac{4}{15}\sin{\theta}(1+4\cos{\theta}) = \mp\Theta
\end{equation}
\hrule\vspace{5mm}

Видно, что надо рассматривать два разных случая:
с одинаковыми и с разными знаками у $n_y$ и $\theta$.

\begin{equation}
\left\{
\begin{array}{ccl}
0 &=&   h m_z  +  d m_x \mp b\Theta\\
0 &=&  2(1+d) + (m_z-1) \mp \left(\sqrt{\frac35} - \frac45\Theta\right) (m_x-h)\\
0 &=&  (m_x-h) \pm \left(\sqrt{\frac35} - \frac45\Theta\right)(m_z-1)\\
\end{array}\right.
\end{equation}


Из второго уравнения
\begin{equation}
m_z =  - 1 - 2d  \pm\sqrt{\frac35}(m_x-h) \mp \frac45\Theta m_x
\end{equation}

\begin{equation}
\left\{
\begin{array}{ccl}
m_x (d \pm \sqrt{\frac35}h) &=& h \pm b\Theta\\
m_x (1-\sqrt{\frac35} \Theta) &=& h
\pm \frac{\sqrt{15}}4 (1+d)
\mp \Theta
\end{array}\right.
\end{equation}

\begin{equation}
\frac{h \pm b\Theta}{d \pm \sqrt{\frac35}h}(1-\sqrt{\frac35} \Theta) =
 h \pm \frac{\sqrt{15}}4 (1+d) \mp \Theta
\end{equation}

\begin{equation}
h \pm b\Theta = \pm \frac{\sqrt{15}}4\left(d \pm \sqrt{\frac35}h\right)
\end{equation}

\begin{equation}
\Theta = \frac1{4b}\left(\sqrt{15}d \mp h\right), \qquad
m_x  = h \pm \frac{\sqrt{15}}4 (1+d) \mp \Theta, \qquad
m_z  = - \frac14 - \frac54 d - 2\sqrt{\frac35} \Theta
\end{equation}
