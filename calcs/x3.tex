\section*{3. Стационарное решение с учетом поглощения}

\hrule\vspace{5mm}

Стационарные уравнения Леггетта-Такаги во вращающейся системе координат:

\begin{equation}
\left\{
\begin{array}{ccl}
0 &=& - d m_y + bF n_x\\
0 &=& h m_z + d m_x + bF n_y\\
0 &=& - m_y h + bFn_z\\
0 &=& B + kF\\
0 &=& - 2n_y(1+d) - n_y(m_z-1) + n_zm_y - C Bn_x + C(m_x-h)\\
0 &=&   2n_x(1+d) - n_z(m_x-h) + n_x(m_z-1) - C Bn_y + Cm_y\\
0 &=& - n_xm_y + n_y(m_x-h) - C Bn_z + C(m_z-1)\\
B &=& n_x(m_x-h) + n_ym_y + n_z(m_z-1)
\end{array}\right.
\end{equation}

Для отклонений от случая без поглощения:

\begin{equation}
\left\{
\begin{array}{ccl}
0  &=& - d \Delta m_y - b\Theta_0 \Delta n_x\\
0  &=&   d \Delta m_x + h\Delta m_z -
  b\Theta_0 \Delta n_y - b\Delta\Theta\\
0  &=& - h \Delta m_y - b\Theta_0 \Delta n_z\\
0&=& \Delta m_y + \frac{\sqrt{15}}{4}\Delta n_x -
  \frac54\Delta n_z - k\Theta_0\\
0 &=& \sqrt{\frac35}\Delta m_x - \Delta m_z -
  \frac34 \Delta n_y - \sqrt{\frac35}\Delta\Theta\\
0 &=& \Delta m_x + \sqrt{\frac35}\Delta m_z +
  \frac{\sqrt{15}}{4} \Delta n_y + \Delta \Theta\\
\end{array}\right.
\end{equation}

Из первого и третьего уравнения:

\begin{equation}
\Delta n_x=-\frac{d \Delta m_y}{b\Theta_0},\qquad
\Delta n_z=-\frac{h \Delta m_y}{b\Theta_0}.
\end{equation}

Подставим в четвертое уравнение:

\begin{equation}
0 = \Delta m_y - \frac{\sqrt{15}}{4}\frac{d \Delta m_y}{b\Theta_0}
	+ \frac54\frac{h \Delta m_y}{b\Theta_0} - k\Theta_0
\end{equation}

\begin{equation}
\Delta m_y = k\frac{(\sqrt{15}d-h)^2}{16hb} =
 \frac{kb\Theta_0^2}{h}
\end{equation}

Отсюда
\begin{equation}
\Delta n_x=-\frac{d}{h}k\Theta_0,\qquad
\Delta n_z=-k\Theta_0.
\end{equation}

Из условия $n_x^2 + n_y^2 + n_z^2 = 1$

\begin{equation}
\Delta n_x^2 + \Delta n_z^2 + 2\Delta n_y = 0, \qquad
\Delta n_y = -k^2\Theta_0^2 \frac{1+(d/h)^2}{2}.
\end{equation}

Из оставшиеся трех уравнений видно, что и
$\Delta m_x, \Delta m_z, \Delta \Theta \sim k^2\Theta_0^2$

\vspace{5mm}
\noindent\fbox{\parbox{\textwidth}{
Добавки, связанные с поглощением, без учета членов $\sim k^2\Theta_0^2$:
$$
\Delta n_x=-\frac{d}{h}k\Theta,\qquad
\Delta n_z=-k\Theta,\qquad
\Delta n_y =0,
$$
$$
\Delta m_y = \frac{kb\Theta^2}{h},\qquad
\Delta m_x=\Delta m_z = 0,\qquad
\Delta \Theta = 0.
$$
